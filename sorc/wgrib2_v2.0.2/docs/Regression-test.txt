How to run verification:

1. Copy canned data file into verification directory.

2. Copy verify2.sh script into verification directory.

3. Copy new wgrib2 executable into verification directory as wgrib2new.

4. Generate canned RESULTS file for CHKSUM comparison with original wgrib2 (need to be newer version of wgrib2 then the one that is on nwprod as of 2015-09-10). Can just use the RESULTS file from current SVN tag.
./verify2.sh SAVE        
  Create a RESULTS file by running original wgrib2 file against canned data:
    ds.td.bin
    FR_land_cosmo_de.grib2
    gep19.t00z.pgrb2af180
    grid221.g2
    NGAC2d.GrbF00
    png.grb2

5. Run test.
./verify2.sh <RESULTS
